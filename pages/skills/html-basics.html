HTML Basics

This page introduces the basics of Hypertext Markup Language (HTML), the language that structures content on the web.

Foundations

Basic Tags
<!DOCTYPE>Defines the HTML document type
<html>Root element of the HTML page
<head>Container for meta info (like charset, title)
<body>Visible page content
<title>Sets the browser tab title
Text
<h1>â€“<h6>Headings from largest to smallest
<p>Defines a paragraph of text
<br>Line break (empty element)
<hr>Horizontal rule (divider)
Links & Media
<a>Creates a hyperlink
<img>Displays an image
<video>Embeds a video
<audio>Embeds audio
Lists
<ul>Unordered list (bullets)
<ol>Ordered list (numbers)
<li>List item in a list
Tables
<table>Defines a table
<tr>Table row
<td>Table cell
<th>Table header cell
Forms
<form>Form container
<input>Input field (text, checkbox, etc.)
<label>Label for form input
<textarea>Multi-line text box
<select>Dropdown menu

Basic Page Structure

Every HTML page starts with this structure:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>
