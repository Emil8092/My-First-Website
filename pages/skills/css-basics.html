CSS Basics

CSS (Cascading Style Sheets) controls how HTML elements look. Below are foundational concepts.

Foundations

Selectors
elementTargets HTML tags (e.g., p)
.classTargets class attributes
#idTargets unique element IDs
Colors
colorSets text color
background-colorSets background color
HEX, RGBColor formats (e.g., #fff, rgb(0,0,0))
Text Styling
font-sizeChanges text size
font-familySets the font type
text-alignAligns text (e.g., center, left)
Box Model
marginSpace outside an element
paddingSpace inside an element
borderBorder around the element
width, heightDimensions of elements
Layout
displayBlock, inline, flex, grid, etc.
flexboxPowerful 1D layout system
grid2D layout system for rows and columns
positionRelative, absolute, fixed, sticky
Pseudo-classes
:hoverWhen mouse hovers
:focusWhen input is active
:first-childTarget first element
Media Queries
@mediaMake responsive layouts
